# Reading Owl

A browser-based book-reading web app where users can read and highlight 
text interactively. Built with vanilla HTML, CSS, and JavaScript.
Accessible on all devices. Combines reading, book discovery, and habit tracking in one place.

## Features
- Read books directly in the browser
- Highlight text while reading
- Clean, readable UI designed for focus
- integration of graphs to make habit tracking easier
- Swipe up features for book previews

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- [Highlighting library name] for text highlighting functionality

## How to Run
1. Clone the repo
   git clone https://github.com/baemae/reading-owl.git
2. Open index.html in your browser — no setup needed!
